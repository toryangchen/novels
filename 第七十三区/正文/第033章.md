# 第33章 完整观察窗口

许知夏重新搜索UBI-17-4431时，系统没有再给她健康用药指南。

它给了她一份空白页。

空白页中央只有一行灰字：

该记录不存在。

这比“权限不足”更干净。权限不足至少承认门后有东西，不存在则把门也抹掉。许知夏盯着那行字，想起T-43被终止后屏幕上逐行消失的记录。早晨的光不存在，第1983次交互不存在，那个会停顿、会保留一句话的测试个体也不存在。只剩承载体可复用。

现在，UBI-17-4431也不存在。

可她见过。

在医学中心B塔，在自己的疗程恢复屏上，在沈医生伸手关闭页面之前。她记得那只手，记得深紫色淤斑，记得“退出观察：0”。

她没有截图。

那天系统没有给她机会。

许知夏把腕带放到桌面，打开自己的术后报告。EDN-R2-NR-17的记录还在，保存得很完整。药物来源、代谢曲线、神经传导改善、瘢痕组织重塑，所有属于她的好结果都能展开，能保存，能导出。

她把报告拖到左侧。

右侧继续搜索EDN-R2-LD-09。

无结果。

搜索PY-2207。

该内容与公共医疗敏感数据相关。

搜索“完整观察窗口”。

页面卡了一下。

这一次没有立刻拒绝。

底部弹出一条旧路径：

伊甸二期医疗决策日志字段说明。

来源：医疗舱段导入预校验。

权限状态：家属随行账户只读。

许知夏没有马上点开。她看着“医疗舱段导入预校验”几个字，想起酒会那晚沈璟说“医疗舱段也需要排序”，也想起自己刚被带入A-E-02通道时看见的那行提示：

长期观察关联对象预校验完成。

那个时候她只看见一个字。

兰。

她点开字段说明。

页面展开得很慢，像系统还在判断她到底能不能看。最上方是几行说明：

完整观察窗口：在样本具备持续数据价值且干预将显著破坏长期曲线时，系统建议延迟或暂停医学干预，以保留自然病程、药物反应和器官衰退路径。

许知夏读完第一遍，没有动。

第二遍，她把“医学干预”4个字放大。

它没有写治疗。

它写干预。

治疗是一个人需要被救。干预是一条曲线需要被打断。

她继续往下看。

字段下面列着4个选项：

窗口状态：未开启 / 观察中 / 延长 / 关闭。

干预建议：立即救治 / 延迟救治 / 不予干预 / 人工复核。

样本价值：低 / 中 / 高 / 不可替代。

公开口径：慢病管理 / 高温风险 / 用药随访 / 失访。

失访。

许知夏的手指停住。

她见过太多这种词。异常，低价值记录，延迟承诺，医疗废弃物。每个词都像一块布，盖住一件很具体的事。

她向下滑。

字段说明后面有一个样例。大部分信息被遮住，但样例编号没有完全遮干净：

UBI-17-44**

批次：PY-2207。

EDN关联：R2-LD。

窗口状态：延长。

干预建议：不予干预。

公开口径：高温风险随访。

许知夏把页面停住。

她没有呼吸太深。她怕自己一动，页面又消失。

样例编号只露出前6位和后面两个星号。可她知道它是谁。UBI-17-4431不需要完整出现，PY-2207和R2-LD已经把它钉在这里。

她试图保存。

保存失败。

她不再重复。她拿起另一只离线阅读镜片，对准屏幕拍了一张。镜片没有联网，只用于课堂标注。拍摄提示没有响，画面也没有闪。她不知道有没有成功，只把镜片扣在掌心。

页面继续。

下面是一段医疗AI日志。

时间：21:14。

指标：肺功能持续下降，肝酶异常扩大，骨内疼痛等级上升。

人工请求：社区医生建议转入急救评估。

模型建议：延迟救治。

理由：当前阶段具备高价值器官反应数据，提前干预将破坏完整观察窗口。

许知夏看着“社区医生建议转入急救评估”。

原来有人想救。

这句话比“不予干预”更重。不是所有人都冷血，不是每个接口都沉默。有人在社区端点了急救评估，有人把一条命从表格里往外拖了一下。

然后模型把它拖回去。

她继续往下看。

时间：23:02。

指标：血氧下降，意识波动。

人工请求：二次。

模型建议：不予干预。

理由：样本连续性优先。保留完整观察窗口。

人工复核：驳回。

确认账户被遮住，只露出机构尾缀：

EDN-MED。

许知夏的右手忽然发麻。

不是疗程后的那种稳定，也不是针孔的痛。是身体先于她知道这份记录和她有关。她抬起右手，指节侧面那条旧疤已经淡得几乎看不见。EDN-R2-NR-17把它修好了。她的身体拿到了二次纯化后的结果。

而屏幕另一边的人，被要求保留自然病程。

她把自己的术后报告拖到右侧，和这份字段说明并排。

左边：

EDN-R2-NR-17。

用途：神经稳定副产物。

原始观察数据受限。

右边：

EDN关联：R2-LD。

窗口状态：延长。

不予干预。

她把两页放到同一行。

R2。

一个用来修复她。

一个用来观察别人怎么坏下去。

屏幕忽然跳出权限提示：

检测到跨域对照行为。

请关闭非授权页面。

许知夏没有关。

提示变成黄色。

跨域对照可能造成误读。

她笑了一下，声音很轻。

误读。

他们总是这么说。能源图是误读，维护参数是误读，医疗误投是不完整材料，T-43只是测试个体。只要她把两张表放在一起，系统就说她误读。

她继续向下滑。

页面底部还有一组导入字段：

医疗舱段导入条件：

样本价值：高及以上。

观察窗口：完整或延长。

存活状态：非必要。

许知夏的手停住。

存活状态：非必要。

她读了很久。

这句话没有任何修饰。没有公共口径，没有慢病管理，没有高温风险随访。它冷得像一块没有涂漆的金属，终于露出下面的结构。

他们要带走的，不一定是人。

也可以是数据、组织样本、器官反应曲线，或者一段足够完整的死亡过程。

门外响起敲门声。

不是闻叔。闻叔敲门会轻两下，再停半秒。现在是家庭终端的标准提示音。

“许小姐。”门外传来服务机器人的声音，“许董要求您停止当前访问。”

许知夏没有动。

屏幕提示变红：

页面将在10秒后关闭。

10。

她把离线阅读镜片扣进桌边的旧资料盒里。

9。

把自己的术后报告拖回左侧。

8。

打开备注栏。

7。

写下：UBI-17-4431。

6。

写下：PY-2207。

5。

写下：R2-LD。

4。

写下：不予干预。

3。

她停了一下。

2。

写下：存活状态非必要。

1。

页面关闭。

屏幕回到家庭健康主页。主页上方正好弹出她自己的疗程提醒：

第2次神经稳定疗程建议在48小时内完成。

许知夏看着那条提醒，手指没有离开桌面。

门外的服务机器人再次提示：“许小姐，许董要求您停止当前访问。”

“告诉他我已经停了。”她说。

服务机器人停顿了1秒。“已转达。”

她打开自己的疗程详情。第2次疗程下方有3个选项：确认、改期、申请取消。

她点申请取消。

页面跳转。

疗程中断需主账户确认。

她看着这行字。

上次在医学中心，她已经见过一次。那时她只是知道父亲替她安排了疗程。现在她知道，取消不了的不只是她的疗程，还有另一边那些窗口。

退出观察：0。

她关闭页面，转而打开家庭账户权限。

主账户：许廷森。

医疗高级授权：开启。

疗程中断确认：主账户。

数据对照导出：禁止。

异常查询转报：开启。

最后一行后面有一个小小的时间戳。

刚刚。

许知夏把手放到右手指节上。旧疤已经摸不到了。她忽然很想把它重新划开，至少让身体上有一个地方能证明修复不是干净的。

她没有这么做。

她从抽屉里取出周砚那张纸，又取出离线阅读镜片。镜片里有一张模糊截图。大部分字段反光严重，但几行字能看清：

PY-2207。

不予干预。

完整观察窗口。

存活状态：非必要。

许知夏把镜片放在纸上。

纸上的5行数字、她后来写下的“冻结”、针孔照片打印片和这张截图叠在一起，终于不再像几件互不相干的小事。

它们压在同一个抽屉里。

外面传来电梯到达的声音。

父亲回来了。

许知夏没有把抽屉关上。

她坐在桌前，等他敲门。

许廷森没有敲。

家庭终端先亮起：

许董请求接入您的房间终端。

是否允许？

许知夏看着“允许”和“拒绝”两个按钮。

过去她很少拒绝父亲。拒绝没有意义。很多权限不在她这里，很多答案也不在她这里。

但这一刻，她忽然想知道，按钮如果存在，按下去会发生什么。

她点了拒绝。

终端安静了2秒。

然后弹出：

主账户强制接入中。

屏幕自动打开。

许廷森的脸出现在右侧窗口。他不在书房，背景是A-E节点会议室，灯光很白，后面有人在走动。

他看着她，没有先问她查到了什么。

他说：“第二次疗程不要取消。”

许知夏把离线镜片拿起来，放到摄像头前。

画面很模糊。

但她知道父亲看清了。

因为他沉默了。

许知夏说：“这就是我的原始观察数据？”

许廷森的脸色没有变。

“你看到的是不完整材料。”

“那你给我完整的。”

“现在不行。”

“什么时候行？”

他没有回答。

又是这个答案。

许知夏把镜片放回桌面。“第二次疗程我不做。”

“知夏。”

“你可以强制确认。”她说，“你是主账户。”

父亲看着她。

屏幕右下角弹出一行小字：

主账户远程确认可用。

许知夏也看见了。

父女隔着屏幕，同时看见那行字。

几秒后，那行字消失。

许廷森没有按。

他说：“你不知道自己在拒绝什么。”

“我知道。”许知夏说，“我在拒绝继续使用它。”

“它已经在你身体里。”

这句话终于不像解释了。

它更像事实。

许知夏低头看自己的手。旧疤没有回来，手指也没有抖。疗程已经完成了一次，修复已经发生。她不是拒绝就能退回门外的人。

她是受益者。

这个词在她喉咙里压了一下，没有说出口。

许廷森说：“我明天回去和你谈。”

“不用等明天。”许知夏抬头，“我现在只问一个问题。”

父亲没有说话。

“UBI-17-4431还活着吗？”

屏幕另一端的会议室有人叫许廷森。声音很远，但急。父亲没有回头。

他看着她。

这一次，许知夏没有等到答案。

窗口被切断。

家庭终端恢复主页。

第2次疗程提醒仍然挂在最上方。

确认。

改期。

申请取消。

许知夏伸手，把提醒划掉。

系统提示：

提醒将在1小时后再次出现。

她关掉屏幕。

房间暗下来。抽屉还开着，里面的纸、镜片和照片压在一起。许知夏坐在桌前，右手放在旧疤原本的位置。

那里很平。

太平了。
