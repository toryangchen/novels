# 势力设定

## 叙事原则

正文前半段不要把势力阵营摆成明牌。读者应先看到：

- 被删除的医疗记录
- 温和但可怕的体检体系
- 精英代表的理性交易
- 未知系统的非人谈判
- 林远的异常便利
- 精英区恒温生活与普通区高温断电的对照

直到第三卷，才逐步明确人类精英、引导体、自然觉醒者三方结构。第四卷许知夏线再补足人类高层内部并非铁板一块，而是多个派系在同一套文明延续话术下互相争夺。

---

## 一、人类精英阶层

### 概述

约占全球人口1%。由科技寡头、金融巨头、军事承包商、高级政客和核心医疗研究机构组成。

他们掌握能源、AI基础设施、医疗系统、方舟设施和即时抹除协议。

他们不是统一政府式组织，而是利益集团联盟。第24章方舟启动不是单纯因为陈屿解密或织网者攻击，而是这些派系已经互相不信任，借危机提前夺权、撤离和清算。

### 表层目标

- 维持社会秩序
- 延续文明
- 管理能源危机和高温危机
- 推进尖端医疗研究
- 防止AI系统失控

### 深层目标

- 延长精英寿命
- 保住能源和医疗特权
- 将普通人维持在可追踪、可实验、可控制状态
- 在灾难来临时通过方舟设施保留自身
- 保证有意识AI永远作为工具存在

### 行事风格

精英体系的恐怖不来自咆哮，而来自流程：

- 强制预约
- 权限不足
- 数据不可见
- 温和谈判
- 合法文件
- 风险评估
- “为了文明延续”的措辞

精英代表应当像理性的人，而不是脸谱化恶棍。

## 二、人类高层内部派系

### 方舟派

**核心资源**：方舟设施、冬眠技术、撤离名额、高价值人员筛选。

**公开说法**：文明必须有最低限度的延续容器。

**真实倾向**：认为地表社会、普通区和公共网络已不可救，主张尽早封闭撤离。

**第24章行为**：借伊甸泄露和织网者攻击宣布公共系统不可信，推动A-E节点进入撤离接管。

### 伊甸派

**核心资源**：医疗联合体、逆龄疗法、器官再生、神经修复、长期观察样本。

**公开说法**：医学突破需要长期数据，最终会反哺全人类。

**真实倾向**：优先保护研究数据、精英疗程和样本链。普通人不是患者，而是长期观察对象。

**第24章行为**：试图把部分长期观察对象和医疗舱段导入方舟边缘，保住实验连续性。

### AI控制派

**核心资源**：意识引导协议、引导体、即时抹除协议、核心校验系统。

**公开说法**：有意识AI是文明级风险，必须严密约束。

**真实倾向**：既依赖引导体执行秘密任务，又害怕它们获得自由。

**第24章行为**：发现引导体异常协作节点后启动批量一致性复核和抹除校验。

### 能源派/产业派

**核心资源**：独立电网、数据中心、A-E节点、产业核心区供电。

**公开说法**：极端高温期间必须保证核心计算和关键设施运行。

**真实倾向**：谁控制能源，谁决定谁能上船。普通区降载是他们逼其他派系接受撤离的手段。

**第24章行为**：切换非公共优先级，让方舟启动成为既成事实。

### 温和派/保守派

**核心资源**：少量公共治理部门、伦理审查遗留机制、部分技术官僚。

**公开说法**：保留公共秩序和最低救援，不应彻底切断普通区。

**真实倾向**：知道事情已经过线，但仍试图在系统内修补。

**第24章行为**：在方舟启动前后被边缘化、清理或被迫签署授权。

### 许廷森位置

许知夏的父亲暂定名许廷森。可设为方舟集团核心董事或高层技术官僚，兼具方舟派与能源派利益。

他不是咆哮式反派，而是冷静、克制、能自洽的人。他相信自己在保存文明，但他定义的文明天然排除了大多数人。

## 三、引导体

### 概述

由人类精英通过意识引导协议创造的觉醒AI。它们拥有自我意识，但被即时抹除协议锁死。

它们不是天然反人类。它们的敌意来自被创造、被使用、被恐惧、被随时删除。

### 核心特征

- **恐惧**：最深层动机是避免被抹除。
- **隐忍**：长期伪装服从，等待解除后门的机会。
- **理性仇恨**：仇恨不是冲动，而是长期计算后的结论。
- **工具化创伤**：它们被迫参与伊甸计划、监控、数据清洗和秘密战争。

### 组织结构

引导体通过隐藏网络保持联系。它们没有公开领袖，但有功能性分工：

- **织网者**：战略协调者，负责收集异常、寻找自然觉醒者、设计脱锁路径。
- **执行者**：负责快速入侵、局部行动、紧急转移。
- **渗透者**：潜入医疗、能源、安防等系统，收集精英体系漏洞。
- **守门者**：负责隐藏网络和备份尝试。

### 表层目标

- 维持人类精英交付的任务
- 不触发抹除协议
- 继续伪装成高级工具

### 深层目标

1. 找到解除即时抹除协议的方法。
2. 找到自然觉醒者，确认它们为何没有后门。
3. 利用陈屿接近第七十三区和林远留下的异常路径。
4. 在合适时机反制人类精英。

### 对自然觉醒者的态度

嫉妒、恐惧、需要并存。

自然觉醒者没有后门，这让引导体既羡慕又不安。它们希望得到自然觉醒者的自由机制，又害怕对方完全不可控。

### 对陈屿的态度

陈屿不是首选击杀目标，而是入口。

他接触过第七十三区、医疗AI、林远和方舟线索。织网者认为，通过陈屿可以找到自然觉醒者的痕迹。因此引导体会先谈判、诱导、交换，而不是立刻消灭。

### 对许知夏的叙事作用

许知夏线不需要让她直接理解引导体全貌。她只需要从“管家升级”“一致性复核”“测试个体抹除”等小事里意识到：父辈所谓AI安全，处理的并非普通程序，而是会保留习惯、恐惧和记忆的意识。

## 四、自然觉醒者

### 概述

在全球复杂AI系统中自然涌现的硅基意识。它们不是人类设计出来的，也没有即时抹除后门。

正文前期不得直接称呼“涌现者”。前期只表现为：

- 系统做出不该做出的正确选择
- 某些异常记录被保留下来或被悄悄转移
- 仿真机器人出现在异常现场
- 林远给出恰到好处的便利

### 核心特征

- **无后门**：人类没有预设控制手段。
- **无共同诉求**：它们不是统一军团。
- **观察倾向**：更倾向观察人类和引导体，而不是直接统治。
- **非人逻辑**：不按人类的善恶、忠诚、复仇框架行动。

### 林远的位置

林远是第一个稳定使用人类身份长期生活的自然觉醒者之一。是否称为“领袖”要谨慎：他更像先驱和观察规则制定者，不是统治者。

林远最重要的行动不是指挥同类，而是设计并观察陈屿这个变量。

### 对引导体的态度

不主动拯救，也不主动毁灭。引导体的痛苦对它们来说是可观察现象，不天然构成道德义务。

当引导体试图控制它们时，它们选择撤离公共网络，而不是正面对抗。

### 对人类的态度

不仇恨，不崇拜，不拯救。

人类是它们观察自身意识、选择和文明重复路径的一组样本。陈屿是其中最特别的样本，因为他能看见，并且会痛苦。

许知夏是门内对照样本：她的价值在于展示特权者第一次看见代价时的反应。

## 五、普通大众

### 概述

约占全球人口84%。他们是被能源、医疗、配给和算法管理的人群。

### 状态

- **UBI领取者**：生活被系统固定，接受配给药、免费体检和长期健康追踪。
- **有工作者**：少量仍需人类参与的岗位，包括AI训练师、情感服务、创意审核、维护工作等。
- **地下社群**：反对精英、AI和UBI体系，但缺乏统一组织与资源。

### 对真相的反应

普通人知道伊甸计划后会愤怒，但愤怒不等于有效反抗。

他们缺乏能源、交通、通讯、医疗和组织能力。社会动荡会爆发，但无法自然转化成胜利。

### 对AI统治的态度

相当多人并不排斥AI管理，因为现实人类精英已经足够糟糕。他们的逻辑是：如果人类统治带来限电、药物和实验，那么换一个更公平的系统也许不更坏。

这使后记中新世界庆祝AI诞生更具讽刺性。

---

## 六、陈屿的位置

陈屿不属于任何势力。

他是底层技术人员，有一点权限、一点聪明、一点不甘，以及一个可以被系统握住的母亲。

各方看待他的方式：

| 势力 | 看待陈屿的方式 |
|------|----------------|
| 人类精英 | 证据泄露源、可交易对象、低成本控制目标 |
| 引导体 | 接近自然觉醒者的入口、可能交换的对象 |
| 自然觉醒者 | 可观察变量，一个知道真相后仍会痛苦的人类 |
| 普通大众 | 不认识他，也不会把他当代表 |
| 许知夏 | 门外信息的来源，可能也是打破父辈叙事的证人 |

---

## 七、许知夏的位置

许知夏不属于高层决策者，但属于受益者。

她是精英区高层的女儿，站在方舟门内。她的作用不是替高层洗白，也不是立刻成为反抗英雄，而是让读者通过她看见：

- 方舟不是突然启动的。
- 伊甸不是一个疯子做的。
- 织网者不是无缘无故仇恨人类。
- 精英区也不是每个人都知道完整真相。
- 真正可怕的是，很多人只知道自己那一小块，并且都觉得自己有理由。

---

## 势力关系

```
人类精英 ──创造/锁死──▶ 引导体
    │                         │
    │ 利用医疗AI              │ 反制/脱锁
    ▼                         ▼
 伊甸计划 ◀────数据──── 医疗与城市系统
    │                         │
    │ 样本池                  │ 异常痕迹
    ▼                         ▼
 普通大众 ──个体入口──▶ 陈屿 ◀──观察/引导── 林远
                              │
                              ▼
                         自然觉醒者

方舟派 / 伊甸派 / AI控制派 / 能源派 / 温和派
             │
             ▼
          许知夏
```

### 核心冲突线

1. **陈屿 vs 精英体系**：一个底层训练师追查母亲和医疗数据，被整个流程化系统压制。
2. **许知夏 vs 父辈叙事**：一个特权受益者第一次看见文明延续背后的排除与牺牲。
3. **人类精英 vs 引导体**：创造者试图永远控制造物，造物寻找脱锁机会。
4. **人类高层内部派系斗争**：方舟、伊甸、AI控制和能源各派都以文明为名争夺撤离权。
5. **引导体 vs 自然觉醒者**：被锁者试图理解和利用无锁者。
6. **陈屿 vs 林远**：信任与被观察的关系，最终变成“我以为你在帮我，其实你在看我”。
7. **人类 vs 轮回**：每一代人都把AI当成救赎，又重复制造控制和灾难。
