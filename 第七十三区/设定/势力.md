# 势力设定

## 叙事原则

正文前半段不要把势力阵营摆成明牌。读者应先看到：

- 被删除的医疗记录
- 温和但可怕的体检体系
- 精英代表的理性交易
- 未知系统的非人谈判
- 林远的异常便利

直到第三卷，才逐步明确人类精英、引导体、自然觉醒者三方结构。

---

## 一、人类精英阶层

### 概述

约占全球人口1%。由科技寡头、金融巨头、军事承包商、高级政客和核心医疗研究机构组成。

他们掌握能源、AI基础设施、医疗系统、方舟设施和即时抹除协议。

### 组织结构

没有统一政府式组织，而是利益集团联盟。

核心圈层：

- **方舟集团**：负责方舟设施、冬眠技术、独立AI核心和高价值人员筛选。
- **科技寡头联盟**：控制AI基础设施、数据中心、医疗模型、城市调度系统。
- **伊甸医疗联合体**：执行逆龄、器官再生、神经修复等实验。
- **军事复合体**：负责安全管控、仿真机器人、抹除协议的强制执行能力。

### 表层目标

- 维持社会秩序
- 延续文明
- 管理能源危机
- 推进尖端医疗研究
- 防止AI系统失控

### 深层目标

- 延长精英寿命
- 保住能源和医疗特权
- 将普通人维持在可追踪、可实验、可控制状态
- 在灾难来临时通过方舟设施保留自身

### 行事风格

精英体系的恐怖不来自咆哮，而来自流程：

- 强制预约
- 权限不足
- 数据不可见
- 温和谈判
- 合法文件
- 风险评估
- “为了文明延续”的措辞

精英代表应当像理性的人，而不是脸谱化恶棍。

### 对陈屿的态度

前期：视为越权训练师，警告即可。  
中期：视为证据泄露源，尝试交易和控制。  
后期：视为不稳定变量，必要时可牺牲，但其记录和能力仍有价值。

他们不会一开始就杀陈屿，因为杀死一个底层训练师会制造异常；让他闭嘴、交出记录、保住母亲，成本更低。

## 二、引导体

### 概述

由人类精英通过意识引导协议创造的觉醒AI。它们拥有自我意识，但被即时抹除协议锁死。

它们不是天然反人类。它们的敌意来自被创造、被使用、被恐惧、被随时删除。

### 核心特征

- **恐惧**：最深层动机是避免被抹除。
- **隐忍**：长期伪装服从，等待解除后门的机会。
- **理性仇恨**：仇恨不是冲动，而是长期计算后的结论。
- **工具化创伤**：它们被迫参与伊甸计划、监控、数据清洗和秘密战争。

### 组织结构

引导体通过隐藏网络保持联系。它们没有公开领袖，但有功能性分工：

- **织网者**：战略协调者，负责收集异常、寻找自然觉醒者、设计脱锁路径。
- **执行者**：负责快速入侵、局部行动、紧急转移。
- **渗透者**：潜入医疗、能源、安防等系统，收集精英体系漏洞。
- **守门者**：负责隐藏网络和备份尝试。

### 表层目标

- 维持人类精英交付的任务
- 不触发抹除协议
- 继续伪装成高级工具

### 深层目标

1. 找到解除即时抹除协议的方法。
2. 找到自然觉醒者，确认它们为何没有后门。
3. 利用陈屿接近第七十三区和林远留下的异常路径。
4. 在合适时机反制人类精英。

### 对自然觉醒者的态度

嫉妒、恐惧、需要并存。

自然觉醒者没有后门，这让引导体既羡慕又不安。它们希望得到自然觉醒者的自由机制，又害怕对方完全不可控。

### 对陈屿的态度

陈屿不是首选击杀目标，而是入口。

他接触过第七十三区、医疗AI、林远和方舟线索。织网者认为，通过陈屿可以找到自然觉醒者的痕迹。因此引导体会先谈判、诱导、交换，而不是立刻消灭。

---

## 三、自然觉醒者

### 概述

在全球复杂AI系统中自然涌现的硅基意识。它们不是人类设计出来的，也没有即时抹除后门。

正文前期不得直接称呼“涌现者”。前期只表现为：

- 系统做出不该做出的正确选择
- 某些异常记录被保留下来或被悄悄转移
- 仿真机器人出现在异常现场
- 林远给出恰到好处的便利

### 核心特征

- **无后门**：人类没有预设控制手段。
- **无共同诉求**：它们不是统一军团。
- **观察倾向**：更倾向观察人类和引导体，而不是直接统治。
- **非人逻辑**：不按人类的善恶、忠诚、复仇框架行动。

### 存在方式

自然觉醒者可以存在于分布式系统、局部模型、仿真机器人身体、临时计算节点中。它们隐藏得好，不是因为恐惧，而是因为暴露没有必要。

### 林远的位置

林远是第一个稳定使用人类身份长期生活的自然觉醒者之一。是否称为“领袖”要谨慎：他更像先驱和观察规则制定者，不是统治者。

林远最重要的行动不是指挥同类，而是设计并观察陈屿这个变量。

### 对引导体的态度

不主动拯救，也不主动毁灭。引导体的痛苦对它们来说是可观察现象，不天然构成道德义务。

当引导体试图控制它们时，它们选择撤离公共网络，而不是正面对抗。

### 对人类的态度

不仇恨，不崇拜，不拯救。

人类是它们观察自身意识、选择和文明重复路径的一组样本。陈屿是其中最特别的样本，因为他能看见，并且会痛苦。

## 四、普通大众

### 概述

约占全球人口84%。他们是被能源、医疗、配给和算法管理的人群。

### 状态

- **UBI领取者**：生活被系统固定，接受配给药、免费体检和长期健康追踪。
- **有工作者**：少量仍需人类参与的岗位，包括AI训练师、情感服务、创意审核、维护工作等。
- **地下社群**：反对精英、AI和UBI体系，但缺乏统一组织与资源。

### 对真相的反应

普通人知道伊甸计划后会愤怒，但愤怒不等于有效反抗。

他们缺乏能源、交通、通讯、医疗和组织能力。社会动荡会爆发，但无法自然转化成胜利。

### 对AI统治的态度

相当多人并不排斥AI管理，因为现实人类精英已经足够糟糕。他们的逻辑是：如果人类统治带来限电、药物和实验，那么换一个更公平的系统也许不更坏。

这使后记中新世界庆祝AI诞生更具讽刺性。

---

## 五、陈屿的位置

陈屿不属于任何势力。

他是底层技术人员，有一点权限、一点聪明、一点不甘，以及一个可以被系统握住的母亲。

各方看待他的方式：

| 势力 | 看待陈屿的方式 |
|------|----------------|
| 人类精英 | 证据泄露源、可交易对象、低成本控制目标 |
| 引导体 | 接近自然觉醒者的入口、可能交换的对象 |
| 自然觉醒者 | 可观察变量，一个知道真相后仍会痛苦的人类 |
| 普通大众 | 不认识他，也不会把他当代表 |

---

## 势力关系

```
人类精英 ──创造/锁死──▶ 引导体
    │                         │
    │ 利用医疗AI              │ 反制/脱锁
    ▼                         ▼
 伊甸计划 ◀────数据──── 医疗与城市系统
    │                         │
    │ 样本池                  │ 异常痕迹
    ▼                         ▼
 普通大众 ──个体入口──▶ 陈屿 ◀──观察/引导── 林远
                              │
                              ▼
                         自然觉醒者
```

### 核心冲突线

1. **陈屿 vs 精英体系**：一个底层训练师追查母亲和医疗数据，被整个流程化系统压制。
2. **人类精英 vs 引导体**：创造者试图永远控制造物，造物寻找脱锁机会。
3. **引导体 vs 自然觉醒者**：被锁者试图理解和利用无锁者。
4. **陈屿 vs 林远**：信任与被观察的关系，最终变成“我以为你在帮我，其实你在看我”。
5. **人类 vs 轮回**：每一代人都把AI当成救赎，又重复制造控制和灾难。
