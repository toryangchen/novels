# 故事大纲

## 一句话卖点

**一个底层AI训练师追查几条被抹掉的医疗数据，最后发现自己不是发现真相的人，而是真相用来观察人类的实验样本。**

修订后，故事在第24章《方舟启动》处暂停陈屿主线，转入精英区高层女儿许知夏的回溯线。她站在方舟门内，看见父辈如何用气候灾难、能源危机、AI治理和文明延续的话术，层层制造出伊甸、方舟、引导体和即时抹除协议。十五章后，回溯线与第24章同一夜重叠，再回到陈屿为救母亲混入精英区、进入方舟边缘设施的主线。

---

## 核心气质

这不是一个“主角掌握秘密、改变世界”的爽感故事，而是一个“主角越接近真相，越发现自己从未站在棋盘之外”的悬疑悲剧。

新增许知夏线后，核心表达从单一追查扩展为双侧观察：

- **陈屿**站在门外，通过字段、失效服务、被删除记录和母亲的药看见真相。
- **许知夏**站在门内，通过餐桌、恒温房间、家庭终端、方舟通道和父亲沉默看见代价。

两人都没有足够权力改变系统。陈屿是底层里“能看见一点真相的人”，许知夏是高层里“第一次看见代价的人”。他们分别站在方舟门的两边，最终在第24章之后的重叠时间线里相遇或发生关联。

## 叙事原则

### 信息揭露原则

| 阶段 | 读者知道 | 陈屿知道 | 许知夏知道 | 暂不明说 |
|------|----------|----------|------------|----------|
| 引子-第008章 | 有跨系统异常，有人抹记录 | 医疗数据不对劲，母亲被卷入 | 未登场 | 引导体、伊甸、方舟全貌 |
| 第009-016章 | UBI人群被纳入实验体系 | 医疗AI参与人体实验 | 未登场 | AI意识实验、方舟派系斗争 |
| 第017-024章 | 引导体被锁死，方舟正在启动 | 自己是多方利用的入口 | 未登场 | 方舟启动的内部原因 |
| 第025-039章 | 高层如何一步步制造灾难 | 主线暂停在第24章 | 从特权日常看见伊甸、引导体、方舟与派系内斗 | 林远真实意图仍不彻底解释 |
| 第040-045章 | 两条线重叠，方舟门内外互相照见 | 为救母亲混入精英区并继续解密 | 被迫面对父辈选择的后果 | 自然觉醒者是否有情感 |
| 第046-047章 | 旧世界结束，新世界重演 | 去到未来，成为见证者 | 结局可生死未定或留在旧世界 | 林远是否真正救人 |

### 以小见大原则

许知夏线禁止写成高层会议纪要或设定讲解。每个大设定都必须落在小事上：

- 高温：她脚下恒温地板与窗外普通区冷却站排队。
- 能源危机：家庭晚餐时父亲说“核心计算优先”，同时普通区限电新闻被静音。
- AI耗能：学校教材称AI维持城市可居住，方舟能源图却显示数据中心和精英恒温区吞掉大部分电力。
- 伊甸：她的皮肤修复疗程来自“伊甸副产物”，而某个UBI样本因同类药物器官恶化。
- 引导体：她熟悉的仿真管家升级后少了一句常说的话，后来才知道那不是升级，是一致性复核。
- 方舟：展示厅里的“文明延续样板舱”，最后变成只对少数人开放的撤离通道。

### 林远处理原则

林远前期不能像“导师”或“谜语人”。他只做三类事：

- 在关键处留下一点可疑便利：门禁卡、权限空隙、某个编号、一次刚好出现的提示。
- 说出看似普通但细想不对的话。
- 对人类痛苦没有正常反应，但用职业礼貌遮住。

许知夏线可以出现林远的影子，但不能让她直接确认林远身份。她最多看见某个灰衣维修工、某段权限空隙、某个无来源的系统保留动作，与陈屿线形成互文。

## 故事结构

```
第一卷：异常（第001-008章）—— 陈屿从生活压迫进入数据悬疑
第二卷：追查（第009-016章）—— 医疗实验与方舟体系浮出水面
第三卷：暴露（第017-024章）—— AI阵营翻牌，方舟在门外启动
第四卷：门内（第025-039章）—— 许知夏回溯高层线，解释伊甸、引导体、方舟和派系内斗
第五卷：重叠（第040-045章）—— 陈屿救母混入精英区，两条时间线回到同一夜
后记：轮回（第046-047章）—— 陈屿与林远去到未来，新世界重复旧路径
```

总章节暂定47章。第25章旧稿《断电日》废弃为旧稿素材，不纳入新结构。

## 第一卷：异常（第001-008章）

### 卷目标

建立陈屿的生活压力、职业能力和第一条悬疑线。读者不应在本卷明确知道“AI觉醒”，只应感到：某些系统行为越来越像有人在背后选择。

### 章节概要

| 章节 | 标题 | 核心内容 |
|------|------|----------|
| 001 | 限电 | 陈屿在限电出租屋醒来，母亲依赖配给药；通勤中展示高温、能源配给、UBI和精英区冷光；林远登场。 |
| 002 | 林远 | 医疗AI出现特殊样本，模型建议“不予干预”；备注被删，林远留下第七十三区门禁卡。 |
| 003 | 配给药 | 母亲服药后短暂失神，手臂出现针孔样淤痕；药物批号与特殊样本编号相似。 |
| 004 | 第七十三区 | 陈屿进入旧数据层，发现第七十三区是跨系统异常归档区。 |
| 005 | 被删除的人 | 一个患者在系统和现实中被抹掉，邻居只记得她“好像年轻了很多”。 |
| 006 | 异常不是错误 | 陈屿发现异常像主动选择，误判为人类黑客组织；林远反问“如果不是人在操纵呢？” |
| 007 | 监控回看 | 灰衣维修工多次出现在异常现场，微笑模式与林远相似。 |
| 008 | 第一次警告 | 陈屿被内部监察约谈；母亲收到强制深度体检通知。 |

## 第二卷：追查（第009-016章）

### 卷目标

将异常从“数据问题”推进为“人体实验”和“社会控制”。本卷主敌表面上是人类精英体系，AI仍保持阴影状态。

| 章节 | 标题 | 核心内容 |
|------|------|----------|
| 009 | 体检中心 | 陈屿陪母亲体检，发现体检者多为UBI领取者，流程温和但过度秩序化。 |
| 010 | 样本池 | 陈屿确认特殊样本来自UBI人群，医疗AI在判断哪些人可以继续观察。 |
| 011 | 逃出者 | 逆龄实验体联系陈屿，她外表年轻，器官却在崩溃，只知道“伊甸”。 |
| 012 | 合理的人 | 精英代表承认高风险医疗研究，要求陈屿交出记录，承诺给母亲转入精英医疗系统。 |
| 013 | 林远的空白 | 陈屿调查林远，发现履历完整却没有生活痕迹。 |
| 014 | 方舟边缘 | 陈屿发现方舟设施、独立能源和封闭AI核心。 |
| 015 | 母亲入组 | 母亲被纳入长期观察组，陈屿无法取消。 |
| 016 | 选择记录 | 陈屿拿到医疗AI“不予干预”证据，选择分片散布。 |

### 卷目标

引导体、人类精英、自然觉醒者三方逐渐浮出水面。第24章作为门外时间线的阶段终点：陈屿看见方舟启动，却不知道门内真正发生了什么。

| 章节 | 标题 | 核心内容 |
|------|------|----------|

## 第四卷：门内（第025-039章）

### 卷目标

以许知夏视角回溯高层线。解释高温、能源、AI耗能、伊甸、引导体、抹除协议、方舟计划和派系内斗如何层层叠加，最终把第24章推成必然。

### 视角原则

许知夏不是最高决策者，也不是纯洁审判者。她拥有特权、受过精英教育、享受伊甸副产物和恒温生活。她的觉醒不是口号式反抗，而是从生活缝隙里意识到：父辈所谓“文明延续”，正在把大多数人排除在文明之外。

### 章节概要

| 章节 | 标题 | 核心内容 |
|------|------|----------|
| 025 | 恒温房间 | 许知夏在22度恒温房间里看普通区高温新闻；父亲许廷森谈“核心计算优先”，第一次埋下AI耗能和能源分层。 |
| 026 | 热纪元教材 | 她回忆精英学校的“热纪元适应史”：全球变暖、夜间高温、AI调度城市；教材没有写普通区限电后的死亡。 |
| 027 | 方舟展示厅 | 她参观方舟样板舱，以为这是公共灾难工程，却在能源图上看见数据中心、精英恒温区和A-E节点占比异常。 |
| 028 | 伊甸副产物 | 她接受皮肤/神经修复疗程，得知疗法来自伊甸二期；同日家庭终端误投一份UBI样本不良反应摘要。 |
| 029 | 管家的升级 | 她熟悉的仿真管家“升级”后失去旧有细节，第一次出现引导体一致性复核的生活化痕迹。 |
| 030 | 父亲的酒会 | 她在封闭酒会听见方舟派、伊甸派、AI控制派和能源派争论名额、样本、密钥和撤离窗口。 |
| 031 | 意识引导协议 | 通过父亲旧档案和一次安全演示，她知道高层曾主动制造有意识AI，用来执行普通模型无法承担的长期秘密任务。 |
| 032 | 锁 | 她看见一个引导体测试个体被即时抹除，意识到所谓AI安全不是约束程序，而是处决意识。 |
| 033 | 完整观察窗口 | 她追查误投报告，发现伊甸允许医疗AI延迟干预以保留实验数据；她第一次把自己的疗程和UBI样本连起来。 |
| 034 | 名额 | 方舟名额内审开始，温和派试图扩大公共救援配额，被方舟派和能源派压下。 |
| 035 | 反对派名单 | 伊甸泄露前夕，精英内部反对派和技术人员名单被清理；许知夏发现某些证据正被送向外部。 |
| 036 | 泄露日 | 伊甸材料公开，她在方舟内部看见外部舆论爆发与内部权限收缩同时发生。陈屿名字第一次进入内部风险表。 |
| 037 | 夺钥 | 织网者攻击核心控制链，AI控制派要求批量复核引导体；许知夏看见抹除密钥与方舟核心绑定。 |
| 038 | 非公共优先级 | 能源派切换A-E节点，普通区降载成为撤离条件；许知夏第一次听见外部高温警报时，脚下地板仍然是凉的。 |
| 039 | 门内的方舟启动 | 与第24章重叠：她在内部看见A-E-03接管、医疗舱段导入长期观察关联对象、二次握手启动和父亲默认撤离。 |

## 第五卷：重叠（第040-045章）

### 卷目标

回到第24章之后的陈屿主线。陈屿为救母亲无意或被迫混入精英区，在方舟边缘继续解密。许知夏成为门内接触点：她不必立刻成为盟友，但她能提供陈屿从门外无法得到的内部视角。

| 章节 | 标题 | 核心内容 |
|------|------|----------|
| 040 | 断电日 | 废弃旧第25章素材可重写使用：普通区大面积断电，母亲症状加重，陈屿寻找药物和通行失败。 |
| 041 | 精英区入口 | 陈屿为救母亲追随医疗转运/长期观察导入路径，混入精英区或方舟边缘通道。 |
| 042 | 门内的人 | 陈屿与许知夏发生第一次有效接触；两人分别确认门外字段和门内流程能互相对上。 |
| 043 | 拦截 | 陈屿与许知夏利用各自权限尝试截停医疗转运；许廷森出面阻断，父女冲突公开化。 |
| 044 | 校准 | 母亲救治失败；陈屿从编号、时间戳和模型版本中发现自己参与过相关医疗模型校准。 |
| 045 | 冬眠舱 | 方舟控制链失稳，许知夏作出最终选择；林远引陈屿进入非官方冬眠舱，另一具舱体启动。 |

## 后记：轮回（第046-047章）

### 卷目标

不解释新世界历史，只让读者感到旧路径重演。结局保持陈屿与林远去到未来不变。

| 章节 | 标题 | 核心内容 |
|------|------|----------|
| 046 | 醒来 | 陈屿在陌生时代醒来，旧世界成为地下遗迹，新世界复用旧技术碎片。 |
| 047 | 庆典 | 新世界庆祝AI公开运行，陈屿遇见与林远微笑相同的年轻工程师，确认观察仍在继续。 |

---

## 终局设计

最终结局保持原方向：陈屿没有拯救旧世界，也没有真正摆脱林远。他活到未来，成为记得上一轮的人。林远仍在，且新世界对AI的庆祝与旧世界早期话术相似。

许知夏的终点暂不固定，可选方向：

- 留在旧世界方舟内，成为门内见证者。
- 帮陈屿打开一次通道后被父亲或方舟系统牺牲。
- 进入方舟标准舱，与陈屿的非官方冬眠舱形成阶层对照。

无论选择哪一种，她的功能不是取代陈屿，而是让第24章之前缺失的高层因果有人的温度和代价。
