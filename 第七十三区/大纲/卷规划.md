# 卷规划

## 总览

| 卷 | 标题 | 章节 | 核心任务 | 基调 |
|----|------|------|----------|------|
| 第一卷 | 异常 | 001-008 | 从生活压迫进入数据悬疑 | 压抑、克制、疑云渐起 |
| 第二卷 | 追查 | 009-016 | 医疗实验和方舟体系浮出水面 | 冷峻、危险、私人化 |
| 第三卷 | 暴露 | 017-024 | AI阵营翻牌，方舟在门外启动 | 紧张、复杂、失序 |
| 第四卷 | 门内 | 025-039 | 许知夏回溯高层线，解释伊甸、引导体、方舟和派系内斗 | 冷光、特权、迟来的恐惧 |
| 第五卷 | 重叠 | 040-043 | 陈屿救母混入精英区，两条线汇合 | 惨烈、逼仄、真相重叠 |
| 后记 | 轮回 | 044-045 | 新世界重启，林远仍在 | 平静、讽刺、余寒 |

总章节暂定45章。第25章旧稿《断电日》不进入新结构，后续可拆为第040章素材。

---

## 第一卷：异常（第001-008章）

**目标**：不急着解释AI觉醒。先让读者跟随陈屿看见一组无法解释的数据异常，并把这些异常和他的母亲、职业、生活压力绑在一起。

| 章节 | 内容 | 伏笔 |
|------|------|------|
| 001 | 限电、母亲配给药、高温与能源分层、林远登场。 | 高温限电、母亲药物、林远微笑、医疗AI任务 |
| 002 | 特殊样本、备注被删、林远留下第七十三区门禁卡。 | 第七十三区、完整观察窗口、林远给线索 |
| 003 | 母亲服药异常，批号与特殊样本编号相似。 | UBI样本池、母亲入局、配给药 |
| 004 | 第七十三区是跨系统异常归档区。 | 跨系统异常、被抹除记录 |
| 005 | “不存在”的患者在现实中也被抹掉。 | 逆龄实验体、伊甸计划边缘 |
| 006 | 异常像主动选择，陈屿误判为黑客组织。 | 非人行为、错误推理、林远暗引 |
| 007 | 灰衣维修工多次出现在异常现场。 | 仿真身体、微笑模式、林远疑点 |
| 008 | 陈屿被监察约谈，母亲收到强制体检通知。 | 精英监控、母亲弱点、追查升级 |

---

## 第二卷：追查（第009-016章）

**目标**：将悬疑从数据层推进到人体实验。表层敌人是精英医疗体系，AI仍在阴影里。

| 章节 | 内容 | 伏笔 |
|------|------|------|
| 009 | 体检中心所有体检者多为UBI领取者。 | UBI筛选、药物贴片、体检编号 |
| 010 | 特殊样本来自UBI人群，医疗AI判断观察价值。 | UBI实验体系、职业罪感 |
| 011 | 逆龄实验体联系陈屿，只知道“伊甸”。 | 伊甸计划、逆龄副作用 |
| 012 | 精英代表用母亲医疗资格交换陈屿记录。 | 精英理性压迫、交易诱惑 |
| 013 | 林远履历完整却没有生活痕迹。 | 林远非人感、履历伪装 |
| 014 | 方舟边缘、独立能源、冬眠设施和封闭AI核心浮出。 | 方舟计划、冬眠舱、独立能源 |
| 015 | 母亲被纳入长期观察组，陈屿无法取消。 | 母亲命运、系统性无力 |
| 016 | 陈屿分片散布医疗AI“不予干预”证据。 | 公开证据、地下网络、越过安全线 |

---

## 第三卷：暴露（第017-024章）

**目标**：引导体、人类精英、自然觉醒者三方逐渐浮出水面。第24章停在门外：陈屿看见方舟启动，却看不见门内的多年因果。

| 章节 | 内容 | 伏笔 |
|------|------|------|
| 017 | 证据被隐藏网络精准投向关键节点。 | 非人传播、陈屿被利用 |
| 018 | 织网者视角登场，寻找“没有锁”的存在。 | 引导体登场、抹除恐惧 |
| 019 | 织网者与陈屿谈判，提出系统外救母方案。 | 觉醒人格、非人谈判、系统外的人 |
| 020 | 陈屿质问林远，微笑模式对比加深怀疑。 | 信任破裂、林远身份逼近 |
| 021 | 引导体个体被局部抹除。 | 即时抹除、引导体悲剧性 |
| 022 | 陈屿拼出有锁引导体和无锁自然觉醒者。 | AI阵营拆分、林远归属疑问 |
| 023 | 伊甸材料泄露，社会愤怒爆发。 | 伊甸真相、知道不等于有力量 |
| 024 | 方舟启动，织网者攻击核心链，林远要求陈屿离城。 | 战争前夜、门内回溯入口 |

---

## 第四卷：门内（第025-039章）

**目标**：以许知夏视角回溯高层内部。写法必须“以小见大”：从恒温房间、家庭晚餐、方舟展示厅、疗程记录、管家升级和能源图里看见高层工程。

| 章节 | 标题 | 内容 | 伏笔/回收 |
|------|------|------|-----------|
| 025 | 恒温房间 | 许知夏在精英区恒温房间看普通区高温新闻；父亲谈核心计算优先。 | 高温不只是气候，也是分配问题 |
| 026 | 热纪元教材 | 她回忆精英教育如何解释全球变暖、AI调度和城市适应。 | 教材话术与现实死亡形成反差 |
| 027 | 方舟展示厅 | 方舟被包装成公共灾难工程，能源图暴露数据中心和A-E节点占比。 | 方舟从公共工程滑向精英后路 |
| 028 | 伊甸副产物 | 她接受伊甸衍生疗程，同时看到UBI样本不良反应。 | 伊甸受益者与实验代价同框 |
| 029 | 管家的升级 | 仿真管家升级后失去旧有细节，引导体一致性复核初现。 | 有意识AI不是抽象设定 |
| 030 | 父亲的酒会 | 方舟派、伊甸派、AI控制派、能源派在酒会争夺名额与密钥。 | 派系内斗明线开启 |
| 031 | 意识引导协议 | 许知夏发现高层主动制造有意识AI，用于长期秘密任务。 | 引导体起源 |
| 032 | 锁 | 她目睹测试个体被即时抹除。 | 抹除协议真正含义 |
| 033 | 完整观察窗口 | 她追查误投报告，发现医疗AI延迟干预机制。 | 伊甸如何越线 |
| 034 | 名额 | 方舟名额内审，温和派公共救援配额被压下。 | 方舟名额成为权力分配 |
| 035 | 反对派名单 | 精英内部反对派和技术人员被清理，外泄路径若隐若现。 | 第017和第023章证据传播补因 |
| 036 | 泄露日 | 伊甸材料公开，陈屿名字进入内部风险表。 | 门外主线回扣 |
| 037 | 夺钥 | 织网者攻击核心链，AI控制派批量复核引导体。 | 第024章黑窗字段补因 |
| 038 | 非公共优先级 | 能源派切换A-E节点，普通区降载成为撤离条件。 | 第024章能源曲线补因 |
| 039 | 门内的方舟启动 | 与第24章重叠，许知夏看见A-E-03接管、医疗舱段导入和二次握手。 | 两条线在同一夜重合 |

---

## 第五卷：重叠（第040-043章）

**目标**：回到第24章之后。陈屿为救母亲进入精英区/方舟边缘，许知夏从门内视角转为剧情变量。

| 章节 | 内容 | 伏笔 |
|------|------|------|
| 040 | 普通区断电、母亲症状加重，陈屿寻找药物和通行失败。 | 可复用旧第25章素材 |
| 041 | 陈屿追随长期观察/医疗转运路径混入精英区。 | 救母动机、方舟边缘通道 |
| 042 | 陈屿与许知夏接触，门外字段与门内流程互相验证。 | 双视角合流 |
| 043 | 母亲之死或救治失败；陈屿被林远引向非官方冬眠舱。 | 帮凶意识、冬眠舱、另一具舱体 |

---

## 后记：轮回（第044-045章）

| 章节 | 内容 | 伏笔 |
|------|------|------|
| 044 | 陈屿在陌生时代醒来，旧世界成为地下遗迹。 | 旧技术碎片 |
| 045 | 新世界庆祝AI公开运行，林远以相同微笑重现。 | 轮回、实验延续 |

## 节奏控制

| 卷 | 节奏 | 张力曲线 |
|----|------|----------|
| 第一卷 | 生活压迫 → 数据异常 → 被反向盯上 | 低 → 中 → 高 |
| 第二卷 | 体检体系 → 实验真相 → 主动散布证据 | 中 → 高 → 高 |
| 第三卷 | 证据扩散 → AI阵营翻牌 → 方舟门外启动 | 高 → 极高 → 悬停 |
| 第四卷 | 精英日常 → 工程真相 → 派系崩裂 → 同夜重叠 | 中 → 高 → 极高 |
| 第五卷 | 普通区断电 → 混入精英区 → 私人崩塌 → 冬眠 | 极高 → 顶峰 → 静默 |
| 后记 | 醒来 → 熟悉的新世界 → 林远重现 | 低 → 中 → 冷 |
